山海镇

发表于 2022-03-18 分类于 风水物品

山海镇是一幅占卜图，正中央画八卦太极图，南北各安日与月，中间画三山五岳，四周是五湖四海。

山海镇具有移山排海，将煞气完全排除，颠倒阴阳之功效。所以在风水化煞上应用非常广泛，面对各种形式煞都有化煞效果。镇宅、化煞、旺财、利市、居家平安、事事如意。

材质

木雕、铜质、画于镜面上、书画于纸上。

化解

一般外型煞、天斩煞、官帽煞、路冲、反弓煞、案劫煞、厝角厝刀、路冲、巷冲、破碎煞、凹风煞，有血光车祸之灾；高山大楼尖峰煞、八曜煞、加先后天曜、廿四山曜煞、动土飞土煞、五黄太岁流年煞、卅六天罡、七十二地煞、趋吉避凶、镇押万煞、最妙用。

一般阳宅、商店、公司所犯到，屋角煞、壁刀煞、庙角煞、路冲、巷冲、天斩煞、电线杆、高楼尖峰、刀煞、反光煞、割脚水煞，或左右边所犯到的厝角叫做兽头最凶，流年煞气(五黄、力士、暗剑、大都、小都、太岁、三煞)到，一定伤人命，大则死亡血光，小则病痛不绝，倒闭。

使用方法

  1. 镇宅化煞，通常于用较大的煞气：如开门见山、天斩煞、反弓煞，或同时有二种以上的煞气时使用：如犯官帽煞(主官司缠身)、屋角煞(对内脏造成伤害)、壁刀煞(主血光)、庙角煞、电线杆、高楼尖峰、五黄煞、明堂见大树等。将山海镇正对着形煞按放，安置在窗口或者门前或马路前，对着煞气冲来的地方。

化解天斩煞，两栋大楼间的狭窄空隙，形状如一把刀从半空劈下，在风水上称为「天斩煞」，面对天斩煞的房屋，要留意血光官非、事业不顺、财帛耗损，还要多注意家人健康。将山海镇安置在窗口或门前（面对天斩煞的方向），可收化解之效。

化解尖角煞，现代大楼造型多锐角，经常形成尖角对到门口、窗口容易造成家人健康的不良影响；面对煞气方挂上山海镇，可化解煞气。其它复合型煞气，或门口对车库的开口煞、房屋面对马路或水路的反弓，均可使用山海镇来化解。

  1. 化反弓煞：明堂见水流或马路呈反弓煞，主子孙不孝，面对反弓煞处，挂山海镇化解。

  2. 化凹风煞：住宅外有凹风煞，主血光，神经衰弱及睡眠质量不佳，面对煞气处，挂山海镇化解。

  3. 化巷冲：屋后有巷冲，主犯小人和坐立难安；屋前有巷冲，主血光。面对煞气处，挂山海镇化解。

  4. 璧刀煞：主血光意外之灾，面对煞气切进处挂山海镇化解。

巷冲和璧刀之不同：阳宅受巷冲冲射，只要是居住于此之人都会受影响。而壁刀则会有男女之分，切右边主伤女、切左边主伤男。

  1. 明堂见高楼、高山阻气时，或面对别家之后门背面，或前屋高于己屋时，都可使用山海镇化解。

  2. 明堂见大树，大树冲门之顶心煞、阴气甚重之时使用。

  3. 小人探头煞+电灯柱煞：其阳宅周边有小人探头之煞气，此小人探头因为两边成尖状似将军帽之样。所以为武的小人探头主因小人犯血光意外，而文的小人探头主犯小人易招小偷。可挂山海镇化解。

  4. 家门对着楼梯：公寓大楼内的住宅，常犯住宅大门对到楼梯上下坎，全家大小不安、心神不宁、引起头痛，且容易造成财不留库。在大门上方安置山海镇（约斜45度角）面对楼梯，可化解煞气。

  5. 电线杆+变电器：有血光意外，变电器会让人做出错误判断和决定。
    化解:可以安置山海镇正对煞气安置即可化解。或在靠变电器之墙上钉上4根钢钉用铜线顺时针绕七圈可以化解电磁煞气。

  6. 如果山海镇对面对到别人，建议山海镇搭配放一对铜麒麟，因为铜麒麟是吉祥之动物，且可化解煞气，也可以带来吉祥，只要将麒麟正对煞气之处即可化解煞气。

  7. 家内的房门对准房门，或是房门对到厕所门，会引起家人大小是非争吵、夫妻失和、影响感情，或者门对到厨房的炉灶，妇女多病、财运不顺，可按山海镇在室内的门斗上，可以化解。厅位或店内所犯到的内型煞都可以按制，效果力量无比。

注意

禁忌1、在室内摆放山海镇，不要摆放在卧室，书房等地方。最好是摆放在窗外，大门处，阳台处为好。

禁忌2、山海镇一般是化煞用的，所以最好是摆放在煞位，无煞不要随意摆放。

石敢当

发表于 2022-03-18 分类于 风水物品

石敢当，又称泰山石敢当，中国旧时宅院外或街衢巷口建筑的小石碑。因碑上刻石敢当字样，故名之。作为民间驱邪、禳解方法之一。此风俗始盛于唐代。

关于石敢当的来历，有很多不同的传说。石敢当，又称泰山“石敢当”的文字记载最早见于西汉史游的《急就章》：“师猛虎，石敢当，所不侵，龙未央”。元代陶宗仪《南村辍耕录》中记载“今人家正门适当巷陌桥道之冲，则立一小石将军，或植一小石碑，镌其上曰石敢当，以厌禳之”。

石敢当立于街巷之中，特别是丁字路口等路冲处被称为凶位的墙上。石碑上刻有“石敢当”，或“泰山石敢当”的字，在碑额上还有狮首、虎首等浅浮雕。泰山石敢当习俗从内涵上体现的是“保平安，驱妖邪”，据考证来源自泰安市泰山区邱家店镇前旧县村，现已遍布全国，远播海外。国务院公布首批国家级非物质文化遗产名录，“泰山石敢当”习俗榜上有名。

石敢当的用意从最初的压不祥，辟邪发展到驱风、防水，辟邪，止煞、消灾等多种功效。宋代出土的唐大历五年（770年）的石敢当上刻有“石敢当，镇百鬼，压灾殃，官吏福，百姓康，风教盛，礼乐昌。”文字，可以看出当时石敢当的作用。

南宋后期石敢当完全变成避邪物，使用地点大至河岸、桥头、村镇入口等，功能包括镇百鬼、压灾殃、止风邪、避风煞等。

泰山石说
相传古代认为东岳泰山之石具有独特的灵性和神力，因此泰山石敢当是一种灵石崇拜。传说汉朝时汉武帝登泰山，带回四块泰山石，置未央宫的四角以辟邪。泰山被认为有保佑国家的神功，因此泰山的石头就被认为有保佑家庭的神灵。后来泰山石被人格化，姓石名敢当，又称石将军，后来还发展出了雕刻有人像的石敢当!

泰山石有二十五亿年历史，在所有的石头当中，泰山石的质地是最硬的。古代叫泰山安则天下安，即是它能镇天下，而又能镇乾坤。所以用泰山石来做成的石敢当，镇宅挡煞效果最好。

泰山神说

相传泰山有“主治生死，百鬼主帅也”之称，而中国民间有一种传说是，石敢当是泰山神的部下神将，以勇猛闻名。

石敢当样式

石敢当在不同的地方也有不同的样式，有浅浮雕的；有圆雕的；有的刻有八卦图案；有的什么装饰也没有，只刻有“石敢当”或“泰山石敢当”。

石敢当风水作用

石敢当在风水中起到稳定气场，化煞挡煞的效果比较明显。石敢当是避邪、镇宅、化煞最常使用的风水物品。主要用于挡煞，如道路、河流直冲、箭栋冲射等。

  1. 挡煞
    如住宅大门被道路、河流或屋角等枪煞(路冲、水冲)，或火形尖射煞相冲，房屋周边有垃圾场，房前有教堂、寺庙、公安司法、检察院，以及无法搬动的乱石砖。可在大门口或墙上放置“泰山石敢当”来化解。

  2. 镇宅辟邪
    “泰山石敢当”还是辟邪之物，如果建房地曾为秽地，如垃圾场，医院太平间，墓场等不吉之地，也可用石敢当镇压，以抑秽避凶。

  3. 房屋缺角
    “泰山石敢当”还可化解房屋缺角所带来的煞气，有填实之功效。

  4. 辟邪保平安
    石敢当用在汽车内，可以辟邪保平安。

房屋缺角摆放

距离缺角最近的位置，有有字的一面对着缺角，摆放之前用干净的毛巾擦试一遍。

  1. 东方缺角：
    东为震宫，主是非位，代表长子位置，如果此处缺角既是震挂得不到平衡，容易招致官灾是非，会对家中长子或三十一至四十六岁男士不利，不旺男丁，不利健康事业，且不利属兔的家人。防肝胆方面疾病、足部疾病，并在寅卯辰年流年落空，运程不遂，此年月不利投资、科举、远行。

  2. 东南方缺角:
    东南方为巽宫，代表家中长女之位，家人容易出现呼吸系统疾病，且易出现臀部毛病及坐骨神经痛，且易缺少子嗣，尤以三十一至四十五岁女性，或是生效龙或蛇的女性。如果此处缺角会对家中长女健康学业不利，因东南巽宫五行属木，木在五行中代表肝胆，所以预防肝胆方面疾病，并在辰巳午年流年落空，运程不遂，此年月不利投资、科举、远行。

  3. 南方缺角:
    正南方为离宫，代表中女之位，如果此处缺角会对家中中女及十六至三十岁女子（尤以生效属马者）健康学业不利，因南方为离宫五行属火，需要注意心脏以及眼睛、血液循环、血管等方面的疾病，并在巳午未年流年落空，运程不遂，此年月不利事业发展。

  4. 西南方缺角:
    西南为坤宫，代表家中老母(女主人)之位，如果此处缺角会对家中老母(女主人)及四十六岁以上女性、生肖属羊、猴女性健康家庭不利；坤为土，对脾胃、肠胃方面易发疾病应为注意，并在未申酉年流年落空，运程不佳，在此期间不利于投资发展事业；

  5. 西方缺角:
    正西方为兑宫，代表少女之位，如果此处缺角会对家中十六岁以下少女或生肖属鸡少年健康学业不利；兑金缺少，注意肺和呼吸道系统方面的疾病发生，如口、舌之毛病，喉咙发炎，肺部、呼吸系统不好等，并在申酉戌年流年落空，运程不佳，健康学业有不利影响；

  6. 西北方缺角:
    西北方为乾宫，代表老父(男主人)之位，在风水上乾为天又称为寿山之称，最忌空缺；如果此处缺角会对家中老父(男主人)及四十六岁以上男性或生肖属狗、猪的男性健康事业不利，会造成运气差，阻力大，财来财去等，且易出现头部毛病如高血压、失眠等，并在戌亥子年流年落空；在此年月不利健康、投资、远行、家运不顺；

  7. 北方缺角:
    正北为坎宫，代表中男，在风水上称为财山又为靠山；如果此处缺角会对家中中男及十六至三十岁男性、属鼠男性健康学业不利；会导致运气差，财运衰退，且易出现肾脏、膀胱、利尿系统等毛病及影响女性的子宫，并在亥子丑年落空；注意财运问题；

  8. 东北方缺角:
    东北为艮宫，代表家中少男，在风水上称为子孙山，代表家中人丁旺盛，如果此处缺角会对家中少男或十五岁一下男性及属牛、虎男性和男丁健康学业不利，且易出现关节、烫伤手部等毛病；并在丑寅卯年流年落空，注意脾胃消化系统的疾病发生。

使用方法

凿石起工，选择冬至后的龙虎日，即冬至后甲辰、丙辰、戊辰、庚辰、壬辰、甲寅、丙寅、戊寅、庚寅、壬寅，此十日为龙虎日，用之大吉。先选长石雕刻，除夕夜再以生肉三斤祭拜。

择时安放，新年后，择元月某一吉日的寅时安放，安放时不宜被外人看见。
